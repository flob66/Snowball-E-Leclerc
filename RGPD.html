Politique de confidentialité – Règlement Général sur la Protection des Données (RGPD)

Conformément à nos valeurs, et au Règlement Général sur la Protection des Données (RGPD), nous nous engageons à protéger vos données personnelles que nous recoltons. Le RGPD est un règlement dans la législation de l'UE sur la protection des données et de la vie privée pour tous les individus au sein de l'Union européenne.


Quelles données recueillons-nous ?

- Lors du remplissage des formlulaires :
Votre nom
Votre prénom
Votre photo
Votre adresse IP

- Lors de votre naviguation sur notre site web :
Votre adresse IP


Pourquoi recueillons-nous vos données ?

- Lors de votre visite de notre site Web :
Pour être en mesure de recueillir des informations analytiques qui nous aideront à améliorer notre site Web en général et de cette façon aussi améliorer votre expérience sur le site.

- Lors de l'envoi de formulaires :
Pour être en mesure de vous envoyer votre photo personnalisée avec votre nom et prénom, mais également pour recueillir des informations analytiques nous permettant d'améliorer notre site web.


Combien de temps gardons-nous les données ?

Nous gardons vos données dans une base de donnée sécurisée, pendant une durée de un an. Garder l'historique de nos clients nous permet de créer des données analystique, afin d'améliorer nos services.


Avec qui partageons-nous les données personnelles ?

Nous nous engageons à ne jamais vendre vos données à des tiers. Bien sûr vos informations ne seront jamais dévoilées.


Comment supprimer/modifier vos données personnelles ?

Veuillez nous contacter notre service client.
Toute demande de suppression des données sera traitée sans délai. Nous nous engageons à supprimer toutes données acquises sur le site web vous concernant.


Comment signaler une violation ?

Vous avez le droit de signaler une violation. La violation doit être signalée à l'autorité locale de protection des données (DPA)


Liens externes

Si une partie de ce site Web vous redirige vers d'autres sites, ces sites ne fonctionnent pas dans le cadre de cette politique de confidentialité. Nous vous recommandons d'examiner les déclarations de confidentialité publiées sur ces autres sites Web pour comprendre leurs procédures de collecte, d'utilisation et de divulgation des informations personnelles.
